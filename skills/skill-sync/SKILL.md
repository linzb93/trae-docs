# Skill Sync

一个用于同步项目所需Skills的工具。它会从指定的GitHub仓库中读取技能配置，并根据项目标签筛选并复制相关技能到本地项目。

## 功能

- 读取项目配置获取目标仓库地址和项目标签
- 从目标仓库获取技能列表和标签匹配规则
- 根据项目标签筛选需要的技能
- 将匹配的技能复制到本地项目的 `.trae/skills` 目录

## 使用场景

当项目需要同步更新技能库时，或者在新项目中需要复制其他项目的技能配置时使用。

## 配置要求

项目根目录下的 `secret.json` 需要包含：
- `syncRepoPath`: 目标GitHub仓库地址
- `projectTags`: 项目标签数组

目标项目根目录下的 `compare.json` 需要包含：
- `skills`: 技能列表，每个技能包含 `name`, `url`, `tags` 字段
